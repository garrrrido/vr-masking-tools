# VR Masking tools

Description

## Structure

- `pipeline.py`: Runs the masking pipeline
- `alpha_packer.py`: Packs alpha masks into FISHEYE190 video
- `fisheye190_converter.py`: Converts VR180 video to FISHEYE190 for alpha packing
- `accurate_cut.py`: Cut videos
